I have worked on a number of projects, using multiple languages and frameworks. Here are some of my favorites:

Bright Ideas

Bright Ideas is a website where, once registered and logged in, a user can post any ideas they might have. They can delete their own posts, and they can like other ideas that have been posted. All the ideas are shown on the screen, sorted by the number of likes each one has accumulated. This project was written in C# with the Entity Core Framework, using a MySQL database, and deployed as an EC2 instance using Amazon Web Services.

Github Link: Bright Ideas

Group Chat

This was a simple project that used Socket.IO to create a chatroom. Upon first loading the page, the user was prompted for their name. They could then post messages to the screen that other users could see in real time. Many users could all be chatting with each other simultaneously.

Github Link: Group Chat

Quotable Quotes

This project allowed users to post famous quotes for others to see. A user could delete their own posts, and anyone could select quotes as their favorites. A user's favorite quotes were displayed on one portion of the page, and all the other quotes displayed on a different portion. This project was written in Python, using the Django framework, and a MySQL database. It was deployed as an EC2 instance using Amazon Web Services.

Github Link: Quotable Quotes

Surveys

Surveys was a website that allowed users to post questions with a few different answers, and other users could vote on the option they thought was best. A user could delete their own surveys, but no one elses. This project was made with MEAN (MongoDB, Express.js, Angular, and Node.js) and was deployed as an EC2 instance using Amazon Web Services.

Github Link: Surveys
